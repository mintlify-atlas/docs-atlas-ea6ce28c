---
title: 'Get Active Announcements'
api: 'GET /api/comunicados'
description: 'Retrieve a list of active announcements for the main feed'
---

## Overview

This endpoint returns all active (published) announcements. Inactive or draft announcements are not included in the response. Optionally supports limiting the number of results.

## Query Parameters

<ParamField query="limit" type="integer" optional>
  Maximum number of announcements to return. If not specified, returns all active announcements.
</ParamField>

## Response

<ResponseField name="comunicados" type="array">
  Array of active announcement objects
  
  <ResponseField name="id_comunicado" type="integer">
    Unique identifier for the announcement
  </ResponseField>
  
  <ResponseField name="titulo" type="string">
    Title of the announcement
  </ResponseField>
  
  <ResponseField name="contenido" type="string">
    Content/body of the announcement
  </ResponseField>
  
  <ResponseField name="fecha_publicacion" type="string">
    Publication date in ISO 8601 format
  </ResponseField>
  
  <ResponseField name="activo" type="boolean">
    Whether the announcement is active (always true for this endpoint)
  </ResponseField>
  
  <ResponseField name="media_url" type="string" optional>
    URL to associated media (image, video, etc.)
  </ResponseField>
  
  <ResponseField name="fecha_creacion" type="string">
    Creation timestamp in ISO 8601 format
  </ResponseField>
  
  <ResponseField name="fecha_actualizacion" type="string">
    Last update timestamp in ISO 8601 format
  </ResponseField>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.example.com/api/comunicados?limit=5" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.example.com/api/comunicados?limit=5', {

  }
});

const data = await response.json();
```
</RequestExample>
