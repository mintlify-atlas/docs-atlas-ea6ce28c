---
title: 'Create Announcement'
api: 'POST /api/comunicados/admin'
description: 'Create a new announcement (Admin only)'
---

## Overview

This endpoint allows administrators to create new announcements. Announcements can be created as active (published immediately) or inactive (draft mode).

<Note>
  This endpoint requires admin authentication. Only users with admin privileges can create announcements.
</Note>

## Request Body

<ParamField body="titulo" type="string" required>
  The title of the announcement. This field is mandatory.
</ParamField>

<ParamField body="contenido" type="string" required>
  The content/body of the announcement. This field is mandatory.
</ParamField>

<ParamField body="fecha_publicacion" type="string" optional>
  Publication date in ISO 8601 format. If not provided, defaults to the current timestamp.
</ParamField>

<ParamField body="activo" type="boolean" optional default={true}>
  Whether the announcement should be active (published). Defaults to `true` if not specified.
</ParamField>

<ParamField body="media_url" type="string" optional>
  URL to associated media such as images or videos.
</ParamField>

## Response

<ResponseField name="message" type="string">
  Success message confirming the announcement creation
</ResponseField>

<ResponseField name="comunicado" type="object">
  The newly created announcement object
  
  <ResponseField name="id_comunicado" type="integer">
    Unique identifier for the announcement
  </ResponseField>
  
  <ResponseField name="titulo" type="string">
    Title of the announcement
  </ResponseField>
  
  <ResponseField name="contenido" type="string">
    Content/body of the announcement
  </ResponseField>
  
  <ResponseField name="fecha_publicacion" type="string">
    Publication date in ISO 8601 format
  </ResponseField>
  
  <ResponseField name="activo" type="boolean">
    Whether the announcement is active
  </ResponseField>
  
  <ResponseField name="media_url" type="string" optional>
    URL to associated media
  </ResponseField>
  
  <ResponseField name="fecha_creacion" type="string">
    Creation timestamp in ISO 8601 format
  </ResponseField>
  
  <ResponseField name="fecha_actualizacion" type="string">
    Last update timestamp in ISO 8601 format
  </ResponseField>
</ResponseField>

  }'

import requests

)
