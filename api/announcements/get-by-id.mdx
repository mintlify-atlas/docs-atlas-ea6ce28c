---
title: 'Get Announcement by ID'
api: 'GET /api/comunicados/:id'
description: 'Retrieve a specific active announcement by its unique identifier'
---

## Overview

This endpoint returns a single announcement by its ID. Only active announcements can be accessed through this endpoint. Attempting to retrieve an inactive announcement will result in a 403 Forbidden error.

## Path Parameters

<ParamField path="id" type="integer" required>
  The unique identifier of the announcement to retrieve
</ParamField>

## Response

<ResponseField name="id_comunicado" type="integer">
  Unique identifier for the announcement
</ResponseField>

<ResponseField name="titulo" type="string">
  Title of the announcement
</ResponseField>

<ResponseField name="contenido" type="string">
  Content/body of the announcement
</ResponseField>

<ResponseField name="fecha_publicacion" type="string">
  Publication date in ISO 8601 format
</ResponseField>

<ResponseField name="activo" type="boolean">
  Whether the announcement is active (always true for successful responses)
</ResponseField>

<ResponseField name="media_url" type="string" optional>
  URL to associated media (image, video, etc.)
</ResponseField>

<ResponseField name="fecha_creacion" type="string">
  Creation timestamp in ISO 8601 format
</ResponseField>

<ResponseField name="fecha_actualizacion" type="string">
  Last update timestamp in ISO 8601 format
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.example.com/api/comunicados/1" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.example.com/api/comunicados/1', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.example.com/api/comunicados/1',
    headers={'Authorization': 'Bearer YOUR_TOKEN'}
)

data = response.json()
```
</RequestExample>
